💅 Landing Page - Nail Designer

Este projeto é uma landing page desenvolvida para uma Nail Designer, com foco em apresentação de serviços, estética visual e exibição de feedbacks de clientes.
<br>
<br>
<br>

✨ Objetivo

Criar uma página simples, bonita e funcional para:
<br>

Apresentar os serviços oferecidos
<br>
Gerar interesse em potenciais clientes
<br>
Exibir provas sociais (feedbacks)
<br>
Facilitar o agendamento
<br>
<br>
<br>
🛠️ Tecnologias utilizadas
<br>
HTML5
<br>
CSS3
<br>
<br>
📱 Estrutura da página
<br>
A página é composta por:
<br>
🔝 Header
Nome da profissional
Navegação entre seções (Home, Feedbacks, etc)
<br>
<br>
🎯 Hero Section
Frase principal de impacto
Descrição dos serviços
Botão de ação (agendamento)
Imagem de destaque
<br>
<br>
💬 Feedbacks
Depoimentos de clientes
Imagens ilustrativas
Destaque para qualidade do serviço
